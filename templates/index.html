New Jersey Nitrate Hotspot Research Dashboard

An interactive water-quality research website analyzing nitrate and nitrate + nitrite monitoring records near New Jersey from 2000 to 2026. The project identifies station-level hotspots, compares seasonal and long-term nitrate patterns, and connects watershed hotspots with land use, NJPDES facility density, rainfall, buffer-scale land use, and regression model checks.

About This Project

This website supports a research project on nitrate pollution patterns in and around New Jersey. It combines public monitoring data, spatial watershed boundaries, land-use data, NJPDES facility locations, and model diagnostics to evaluate where hotspots occur and which factors best explain them.

Research Goal

Find where nitrate hotspots occur, test whether they cluster in specific watersheds, and compare developed land, agriculture, facility density, rainfall, and local buffer-scale land use.

Current Finding

Developed land remains the most stable watershed-level predictor after threshold sensitivity, confidence filtering, observation-count weighting, rainfall, buffer-scale, and low-VIF model checks. Facility density is associated with hotspots, but it overlaps strongly with developed land.

Core Watershed Result

Additional Factors: Discharge, Rainfall, Hydrology / Upstream

Step 20 adds advisor-requested factors beyond developed land. Rainfall is estimated at each watershed centroid from daily archive data, hydrology uses watershed shape metrics, and upstream pressure is represented by a same-region upgradient facility-density proxy. Discharge volume is supported when a DMR/discharge CSV is available.

Best added-factor model so far: developed + agriculture + upgradient facility-density proxy. For mean nitrate, adjusted R2 improves to 0.4813 while max VIF remains 1.4835.

Hydrography and DMR Permit-Feature Context

Step 21 reads DMR permit-feature records and NHDPlus HR hydrography. DMR feature numbers are treated as permit-feature / outfall identifiers when available. Because the current DMR and NJPDES facility inputs do not include exact outfall coordinates, the map uses matched NJPDES facility points as clearly labeled outfall proxies.
